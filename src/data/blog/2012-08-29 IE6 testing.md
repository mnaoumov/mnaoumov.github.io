# 2012-08-29 IE6 testing

Unfortunately we still have to support IE6 these days.

For Windows 7 one of the best solutions was [XP Mode](http://www.microsoft.com/windows/virtual-pc/download.aspx)

But this is not longer available in Windows 8.

I ended up installing just real Windows XP on VirtualBox. Needless to mention licensing issues :)

I also found [IE Tester](http://www.my-debugbar.com/wiki/IETester/HomePage) tool useful especially with [DebugBar](http://www.debugbar.com/download.php).

Also I was told about [www.browserstack.com](https://www.browserstack.com) but this did not work properly with my site. The good thing about this service is that they have tunneling and you can easily test local websites. But this service is not free…
