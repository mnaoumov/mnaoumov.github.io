# 2022-05-12 Software for perfectionists

Hi folks.

I am still trying to find the perfect way to get my life organized including all my digital assets into some sort of databases.

I came up using the following software

- **Notes**: [Obsidian](https://obsidian.md/)
- **Photos/videos**: [Adobe Lightroom Classic](https://www.adobe.com/products/photoshop-lightroom-classic.html), [PhotoStructure](https://photostructure.com/)
- **Electronic Books**: [Calibre](https://calibre-ebook.com/)
- **Music**: [MediaMonkey](https://www.mediamonkey.com/)
- **Search files locally**: [Everything](https://www.voidtools.com/)

There are some flaws in all those tools but I could not find better apps.

I still haven't found a good way to organize collection of movies, TV series. I want some tool that can be given a folder and automatically download all possible metadata from IMDB and similar services, store media info, take screenshots etc. All the tools I've tried are not that friendly. But maybe I just don't know how to use them properly. Please share if you are successfully using some.

Also any other ideas for perfectionists are very welcomed.

Stay tuned
