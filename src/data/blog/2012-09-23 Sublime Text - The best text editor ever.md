# 2012-09-23 Sublime Text - The best text editor ever

Previously I heavily used [Notepad++](http://notepad-plus-plus.org/) for about 4 years.

![image-2022-03-27-12-49-25.png](<./!!files/2012-09-23 Sublime Text - The best text editor ever/image-2022-03-27-12-49-25.png>)

But later I found [Sublime Text](http://www.sublimetext.com/) and I fell in love with it. This extremely nice, lightweight and powerful editor ![sublime1.png](<./!!files/2012-09-23 Sublime Text - The best text editor ever/sublime1.png>)

It is a very developers-oriented editor. Press **Ctrl+Shift+P** and you can reach to any available Sublime command ![ctrl-shift-p.png](<./!!files/2012-09-23 Sublime Text - The best text editor ever/ctrl-shift-p.png>)

I recommend to install [Sublime Package Control](http://wbond.net/sublime_packages/package_control) package (plugin) which allows to install new packages very easily.

PowerShell syntax highlighter is not built-in so I install **PowerShell** package.

Having **Sublime Package Control** it is extremely simple. **Ctrl+Shift+P** -> Install package -> PowerShell -> enjoy :)
