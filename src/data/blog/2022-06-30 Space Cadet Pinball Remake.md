# 2022-06-30 Space Cadet Pinball Remake

Hi folks!

One of my favorite computer games is being remade.

For Android you can install apk: https://github.com/fexed/Pinball-on-Android/releases

Improved Windows version: https://github.com/k4zmu2a/SpaceCadetPinball/releases but it requires the data files from the original version, which you can get https://www.groovypost.com/howto/windows-7-3d-pinball-space-cadet-game/

Additionally you can install custom Russian sounds

https://thetrap.ucoz.ru/news/pinball_space_cadet/2010-01-05-24

Feed your nostalgia and enjoy!
