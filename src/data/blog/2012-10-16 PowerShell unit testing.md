# 2012-10-16 PowerShell unit testing

I decided to cover all the hooks I wrote with unit tests. For that I tried to find a good testing framework for PowerShell.

So far, the only one I liked was [Pester](https://github.com/pester/Pester) - it looks nice BDD framework. It has some limitations and I will probably later fork the project to overcome them

I also tried [PsUnit](http://psunit.org/) - was not developed for an year and [PsTest](https://github.com/knutkj/pstest) - did not like its output.
