# 2012-09-24 T4 Syntax highlighting and IntelliSense

Shame on Microsoft but they don't provide any syntax highlighting and IntelliSense for T4 in Visual Studio. There are bunch of 3rd-party utilities for that.

For Visual Studio 2010 the best one is [Visual T4](http://visualstudiogallery.msdn.microsoft.com/40a887aa-f3be-40ec-a85d-37044b239591).

Its free version was even better than some paid products.

Unfortunately I don't find any good utilities for Visual Studio 2012 yet. I will keep you informed
