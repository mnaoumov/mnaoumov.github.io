# 2013-07-09 Sublime Text missing feature - pin tab

I like [Sublime Text](http://www.sublimetext.com/3) editor very much. Recently even bought my own license for it.

One thing is missing for me - is ability to pin tab and prevent it from closing.

[http://www.sublimetext.com/forum/viewtopic.php?f=4&t=493](http://www.sublimetext.com/forum/viewtopic.php?f=4&t=493)

Especially it is useful when I do search in files and want to make a change in all of them. I press Ctrl+W but I can also accidentally close that tab with search results. I want to have an ability to prevent it :)
